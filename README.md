# Painel Administrativo - Sagrada Esperança

Recriado com Nuxt 3 + Tailwind, seguindo os contratos em `API_CONTRATOS.md`.

## Requisitos
- Node.js 18+

## Executar
```bash
cp .env.example .env
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Configuração
- Base URL da API: `NUXT_PUBLIC_API_BASE_URL` no `.env`
